Descripción del Servicio

Automatización y control para upstream, midstream y downstream

Proveemos soluciones tecnológicas especializadas para la industria del petróleo, gas natural y GNL. Nuestros sistemas de control, medición y seguridad cumplen con las normativas más exigentes de la industria energética, garantizando operaciones seguras, eficientes y sostenibles.

¿Qué Incluye Este Servicio?

Soluciones especializadas para cada segmento de la industria

Upstream (Producción)

Control de pozos petroleros y gasíferos, sistemas de separación trifásica, tratamiento de crudo, inyección de agua y gas, medición fiscal de producción.

Midstream (Transporte)

Control de ductos, estaciones de bombeo y compresión, sistemas SCADA para oleoductos y gasoductos, detección de fugas por balance de masa.

Downstream (Refinación)

Control de procesos de refinación, unidades de destilación, cracking catalítico, desulfuración, sistemas de blending y control de calidad de productos.

GNL / Bio GNL

Plantas de licuefacción y regasificación, control de almacenamiento criogénico (-162°C), sistemas de carga/descarga, distribución virtual de GNL.

Sistemas de Seguridad

ESD (Emergency Shutdown) certificado SIL 2/3, sistemas F&G (Fire & Gas Detection), detección de fugas H2S y VOCs, paradas seguras de emergencia.

Medición y Custody Transfer

Sistemas de medición fiscal ultrasónica y tipo coriolis, flow computers, cumplimiento de normas API MPMS, registro de transacciones comerciales.
